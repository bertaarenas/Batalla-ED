# Batalla-ED
Este codigo es una simulación de una batalla entre un enemigo y el jugador. El código cuenta con:
1) Ataque: el jugador intenta hacer daño al enemigo con un 25% de probabilidad de fallar.
2) Defensa: el jugador intenta escapar del daño del ataque enemigo con un 25% de probabilidad de fallar.
3) Estadísticas: Consulta cuando quieras la vida del enemigo y del jugador.
El enemigo aleatoriamente tiene un daño de ataque y tambien se puede defender.
